# FFmpeg Binaries

Esta carpeta debe contener los binarios de FFmpeg para que la aplicación funcione.

## Instrucciones de descarga:

1. Ve a https://www.gyan.dev/ffmpeg/builds/
2. Descarga **ffmpeg-release-essentials.zip** (versión más reciente)
3. Extrae el archivo ZIP
4. Copia `ffmpeg.exe` y `ffprobe.exe` desde la carpeta `bin` del archivo extraído a esta carpeta

## Archivos necesarios:
- `ffmpeg.exe` (~142 MB)
- `ffprobe.exe` (~142 MB)

## Nota:
Los binarios no están incluidos en el repositorio debido a su gran tamaño (>100 MB cada uno, límite de GitHub).
